# Vue + AWS

A VueJS web app running on AWS Cloud Front using Terraform as Infrastructure as Code (IAC)

## Tech Stack

- VueJS (Vite + TypeScript)
- Terraform 
- AWS (S3, CloudFront)

## Local Development

Clone repository

```
git clone https://github.com/EdwinWalela/vueaws
```
Install dependencies
```
npm run install
```

Format Code

```
npm run format
```

Run tests
```
npm run test:unit
```

Start development server

```
npm run dev
```
